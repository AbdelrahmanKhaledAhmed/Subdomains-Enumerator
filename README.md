# Subdomain Enumeration Tool 

Automated subdomain enumeration tool using:

- Subfinder
- Assetfinder
- crt.sh
- httpx
- ntfy notifications

## Features
- Multiple tools integration
- Unique results merging
- HTTP status filtering
- Desktop notifications via ntfy.sh
- Clean terminal UI using Rich

## Installation

1. Install Python 3.10+
2. Install dependencies:
```bash
   pip install -r requirements.txt
```
4. Place required tools in project folder:
   - subfinder.exe
   - assetfinder.exe
   - httpx.exe
   - crtsh.py
   - config.json
